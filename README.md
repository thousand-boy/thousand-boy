## これから制作するポートフォリオ（構想）

### 現場の記録業務を減らすAI活用Webアプリ（1本 / v1 → v2 の二段構え）

作業療法士として6年3ヶ月働くなかで感じた「記録・申し送り・報告書の下書きに時間が取られる」という原体験を出発点に、これから次の構想で制作していきます。

- **課題**：医療・リハビリ現場で、記録や報告書の下書きに時間が取られる
- **v1（最初の目標）**：メモや記録から、要約・下書きを LLM で生成する（FastAPI＋SQLite / React / Render＋Vercel）
- **v2（次の目標）**：現場の文書を取り込み、RAG で出典付きの回答を返す（LangChain・Chroma）
- **大切にしたい点**：見栄えよりも「課題 → なぜその技術を選んだか → つまずきと解決の過程」を言葉にして残すこと
- ※個人情報・医療規制に配慮し、テーマは一般化して制作します

制作の進捗に合わせて、リポジトリと公開URLをこのページに掲載していきます。READMEには、技術を選んだ理由・プロンプト設計・つまずきと解決の過程などを記録していく予定です。

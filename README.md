## 🎯 目指すエンジニア像

業界を問わず、日々の業務をより円滑に進めるための **Web／生成AIアプリケーション** を開発し、現場で本当に使われるプロダクトを生み出していきたいと考えています。

特に関心がある領域：
- 議事録の自動要約・タスク抽出
